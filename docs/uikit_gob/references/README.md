# Referencias visuales — UI Kit v3.0.1

Coloca aquí las **capturas PNG** exportadas desde el kit (o desde Figma) para que queden versionadas en el repo y enlazadas desde [`../DESIGN_SYSTEM.md`](../DESIGN_SYSTEM.md).

## Cómo usar

1. Renombra cada archivo según la tabla (nombres en **kebab-case**, sin espacios).
2. Copia los archivos a esta carpeta: `docs/uikit_gob/references/`.
3. En GitHub o en el IDE, las imágenes se verán en el design system si se usa la sintaxis `![…](uikit_gob/references/nombre.png)` **desde** `docs/DESIGN_SYSTEM.md` (ruta relativa respecto a `docs/`).
